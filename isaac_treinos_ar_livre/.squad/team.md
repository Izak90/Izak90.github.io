## How We Work

- **Decisions:** Recorded in `.squad/decisions.md` (single source of truth)
- **Knowledge:** Each agent has personal memory in `.squad/agents/{name}/history.md`
- **Skills:** Reusable patterns in `.squad/skills/`
- **Logs:** Session and orchestration logs in `.squad/log/` and `.squad/orchestration-log/`

---

## Key Constraints

1. **GitHub Pages Compatible:** No server-side rendering, build tools, or external APIs (unless read-only)
2. **Design System:** Follow existing design tokens, component patterns, and CSS structure
3. **Static Assets Only:** All work must be pure HTML, CSS, JavaScript
4. **Performance:** Optimize bundle size and load times
